# Move Beyond CRUD APIs

As a Backend engineer, learn the following to move beyond CRUD APIs:

---

## 1. API Design
- REST, GraphQL, gRPC
- OpenAPI 3.0 standards
- Status codes, versioning, and pagination strategies

## 2. Security
- **Authentication & Authorization**: OAuth 2.0, JWT, OpenID Connect, Passkeys
- **Cryptography & Encryption**: Algorithms, HTTPS, zero trust
- **OWASP Top 10**
- **Infrastructure**: SIEM, IDS, IPS
- **Web Security**: CORS, Security headers (CSP, HSTS)

## 3. Database Engineering
- SQL & NoSQL
- Query optimization & indexing
- Database trade-offs
- Transactions and isolation levels
- Sharding and partitioning

## 4. Performance
- **Caching**: Redis, CDN, edge caching strategies
- Rate-limiting & Throttling
- Load balancing
- Chaos Engineering
- Fault Tolerance

## 5. Architecture & Paradigms
- Monolith
- Modular monolith
- Microservices
- Serverless vs. Native
- Concurrency & parallelism (reactive programming, structured concurrency)
- Multithreading
- Optimistic and pessimistic locking

## 6. Distributed Systems
- Microservices patterns
- Event-driven architecture (Kafka, Pulsar, RabbitMQ)
- Streaming pipelines
- Service mesh & eventual consistency
- gRPC & Protobuf

## 7. DevOps & Deployment
- CI/CD & GitOps
- Containerization (Docker, Kubernetes)
- Serverless
- Understanding of SLAs
- Incident Management

## 8. Observability
- Logging, monitoring, and tracing (OpenTelemetry)
- Metrics
- Profiling & optimization
- Alerting

## 9. Microservices Patterns
- API Gateway
- Reverse proxy
- Circuit breaker pattern
- Retry/back-off patterns
- Background jobs & scheduling

## 10. AI Integration
- LLM APIs
- Vector databases
- Retrieval-augmented systems (RAG)
